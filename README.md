# SpaceShooter-Game

## About
Space Shooter Game is an action-based arcade game developed using Java. In this game, the player controls a spaceship and shoots enemies while avoiding obstacles and enemy attacks. The objective is to survive for as long as possible, destroy enemy ships, and achieve the highest score. The game provides an exciting gaming experience through interactive controls, collision detection, scoring mechanisms, and graphical effects while demonstrating Java programming and Object-Oriented Programming (OOP) concepts
## Features
-🚀 Spaceship Control – Move the spaceship using keyboard controls.
-🎯 Shooting System – Fire bullets to destroy enemy ships.
-👾 Enemy Generation – Enemies appear continuously during gameplay.
-🏆 Score Tracking – Score increases as enemies are destroyed.
## Technologies Used
Java | Java swing | Oop | Java AWT
